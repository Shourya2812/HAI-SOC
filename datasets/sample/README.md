# Sample Datasets

This directory contains small example datasets generated by the HAI-SOC synthetic log generators.

Contents:

- `ehr_sample.json` – Example EHR audit logs
- `vpn_sample.json` – Example VPN authentication logs
- `firewall_sample.json` – Example firewall network logs

The complete datasets are generated into `datasets/raw/`, which is excluded from Git to keep the repository lightweight.

To regenerate the full datasets:

```bash
python datasets/generators/ehr_generator.py
python datasets/generators/vpn_generator.py
python datasets/generators/firewall_generator.py
```
