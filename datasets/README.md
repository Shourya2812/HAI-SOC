# Healthcare Security Datasets
This directory contains all datasets used by HAI-SOC.


## Structure
datasets/
│
├── generators/
│   Synthetic log generators
│
├── raw/
│   Raw generated logs from different healthcare systems
│
├── normalized/
│   Canonical normalized logs used for ML models
│
├── sample/
│   Small datasets for testing and demonstrations


## Supported Log Sources
- Electronic Health Records (EHR)
- VPN Authentication
- Firewall
- Intrusion Detection System (IDS)
- Medical IoT Devices
- Cloud Audit Logs
